# 🚀 NexaWave Agency

Cartify Ecommerce is a modern and visually appealing landing page designed for an online shopping platform.  
This project focuses on delivering a clean UI, smooth user experience, and a conversion-focused layout that highlights products, offers, and brand identity.

The design is built to simulate a real-world ecommerce experience with attractive sections, responsive design, and engaging interactions.

**View Live Site** 🔗 https://cartify-ecommerce-landing.vercel.app/
